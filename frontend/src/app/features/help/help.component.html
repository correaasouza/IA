support_agent Central de Ajuda
Guia rápido das funcionalidades e como usar o sistema.
Fluxo inicial
1
Selecione o locatário
No login, escolha o locatário permitido. Isso define o contexto do X-Tenant-Id.
2
Cadastre tipos de entidade
Defina Cliente, Fornecedor, Funcionário e os campos obrigatórios.
3
Cadastre registros
Use a tela de Cadastros para inserir registros e contatos.
