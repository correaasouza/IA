# Padroes do Sistema (Frontend)

Documentos oficiais de padrao visual e comportamento:

- `DATE_INPUT_STANDARD.md`: padrao global para campos de data.
- `FORM_FICHA_STANDARD.md`: padrao global para headers e estrutura de fichas.
- `LISTAS_STANDARD.md`: padrao global para bloco superior fixo em listas.
- `ACCESS_CONTROL_DIRECTIVE_STANDARD.md`: padrao global da diretiva de controle de acesso por controle/botao.

Regra obrigatoria complementar:
- listas e fichas devem ser telas separadas (sem formulario inline em listagem).
